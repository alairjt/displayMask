DisplayMask GitHub

What?

Apply a mask in the text.

{{'01234567890' | mask:vm.mask}} ({{vm.mask ? 'With mask' : 'Without mask'}})

How?

In HTML.

{{'01234567890' | mask:'999.999.999-99'}}

{{'01234567890' | mask:'999.999.999-99'}}

In JS.

var text = $filter('mask')('01234567890', '999.999.999-99');
